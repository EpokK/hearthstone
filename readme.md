## Installation

create a `hearthstone` database (`root` / `root`)

get composer : [https://getcomposer.org/doc/00-intro.md](https://getcomposer.org/doc/00-intro.md)

installing dependencies

`php composer.phar install`

migration

`php artisan migrate`

seed

`php artisan db:seed`
